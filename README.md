# Installation
Open terminal on the project directory and enter the below command

```bash
docker-compose up -d
```


Wordpress URL -
```bash
localhost:8000
```

Wordpress wp-admin login => Set up admin password while installing the wordpress

PhpMyAdmin URL
```bash
localhost:8080
```

PhpMyAdmin login
 ```bash
# user
wordpress

# password
wordpress

```
